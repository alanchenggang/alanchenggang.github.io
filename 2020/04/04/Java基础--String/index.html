java.lang.String类代表字符串

API中说:String类代表字符串。 Java程序中的所有字符串文字（例如”abc” ）都被实现为此类的实例。

其实就是说程序中的所有的双引号字符串,都是String类的对象,就算没有new,也照样是.

字符串的特点:

  1. ==字符串的内容永不可变==;
  2. 正是因为字符串不可改变,所以字符串可以共享使用;
  3. 字符串效果上相当于是char[]型字符数组,但是底层原理是byte[]字节数组
