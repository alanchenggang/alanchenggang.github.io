Java基础--Java入门

发表于 2020-04-04 分类于 Java
本文字数： 6k 阅读时长 ≈ 5 分钟

Java是一种可以撰写跨平台应用软件的面向对象的程序设计语言。Java 技术具有卓越的通用性、高效性、平台移植性和安全性，广泛应用于PC、数据中心、游戏控制台、科学超级计算机、移动电话和互联网，同时拥有全球最大的开发者专业社群。

面向对象的三大特征:

​ 封装性/继承性/多态性

基础知识

命名习惯

  • 使用小写字母命名变量和方法，如果一个命名包含多个单词，那么就把它们连起来，第一个单词的字母小写，而后面的每个单词的首字母大写；
  • 类名中的每个单词的首字母大写；
  • 常量中的所有字母大写；

基本语法

数据类型

使用变量的时候的注意事项：

  1. 如果创建多个变量，那么变量的名称不可重复；
  2. 对于float 和 long 类型来说，字母后缀F和L不可丢掉(注意全部用大写)；
  3. 使用byte或者short类型的变量，那么右侧的数据范围不能超过左侧类型的范围；
  4. 没有进行赋值的变量不能直接使用，一定要赋值后才能使用；
  5. 变量的使用不能超过作用域的范围 ；
    作用域：从定义变量的一行开始，一直到直接所属的大括号结束为止。（先创建后使用）
  6. 可通过一个语句创建多个变量，但一般情况下不推荐这么写；

在Java中加号的两种含义：

  • 做加法
  • 做字符串连接符

在源代码中，字符串常量不能跨行，此时加号的第二个作用刚好有用。

数据类型转换

当数据类型不一致时，将会发生数据类型转换；
==字节数与数据范围不相关==

自动类型转换：（隐式转换）

  1. 特点：代码不需要进行特殊处理，自动完成；
  2. 规则：范围从小到大；

强制类型转换：（显式转换）

  1. 代码需进行特殊的格式处理，不可自动完成
  2. 格式： 范围小的类型 范围小的变量名 = (范围小的类型) 原味范围大的数据

数据溢出以及数据类型转换的注意事项

注意事项：

  1. 强制类型转换一般不推荐使用，因为有可能发生精度损失、数据溢出；
  2. byte/short/char 这三种类型都可发生数学运算；
  3. byte/short/char这三种类型在运算时，都会首先提升为int类型，再进行计算；
  4. boolean类型不能发生数据类型转换

常量的命名与使用

语法规则：

习惯上，常量中所有字母都大写。

使用常量的三大好处：

  • 当一个值多次使用时，不必重复输入；
  • 修改常量时比较方便；
  • 给常量赋一个描述性的名字会提高程序的易读性

算术运算符

加号的三种用法

  1. 对于数值来说，就是加法
  2. 对于字符char类型来说，在计算之前，char会被提升为int，然后再计算；char和int之间的对照关系表ASCII Unicode
  3. 对于字符串类型来说，加号表示字符串连接操作

优先级问题

小括号当中的表达式总是优先计算 以多用小括号来代替运算符优先级的定义；

自增自减运算符:

  1. 在单独使用的时候，前++和后++并无区别
  2. 在混合使用的时候，有==重大区别==
    A.如果是==前++==，那么变量==立刻马上+1==，然后拿着结果进行使用；==先加后用==
    B.如果是==后++==，那么优先使用变量本来的值，==然后再让变量+1==；==先用后加==

注意事项：

只有变量才能使用自增自减运算符，常量不能发生改变；

赋值运算符

  1. 只有变量才能使用赋值运算符，常量不能使用；
  2. 复合赋值运算符其中蕴含了一个强制类型转换；

如 byte a = a + 5；等号左边是 byte 类型的变量，等号右边是 byte 类型的变量加上 int 类型的变量

比较运算符

注意事项：

  1. 计较运算符的结果一定是boolean值
  2. 对于数学中 1 < x < 3,程序当中不允许这种写法

逻辑运算符

  1. 与(并且)：全都为true才为true 否则为false
  2. 或(或者)：至少有一个true才为true 全都是false才为false
  3. 非(取反)：本来是true变为false 本来是false 变为 true

与（&&） 或（||）具有短路效应：
如果根据左边的条件即可得知最终结果，则后面的语句不再执行，从而节省一定的性能。

注意事项

  1. 逻辑运算符只适用于boolean 类型的变量
  2. 与、或需要左右各有一个boolean值，但取非仅要一个boolean值即可
  3. 与、或两种运算，如果有多个条件可以连续写

三元运算符

一元运算符： 只需要一个数据就可以进行操作的运算符，例如：取反！、自增++、自减–；

二元运算符：需要两个数据才可以进行操作的运算符，例如加法，减法；

三元运算符：需要三个数据才可以进行操作的运算符。

格式：
数据类型 变量名称 = 条件判断 ? 表达式A : 表达式B;

流程：
首先判断条件是否成立：

如果条件成立为true 那么把表达式A的值赋值给左侧的变量
如果条件不成立为 false 那么把表达式B的值赋值给左侧的变量；
二者选其一

注意事项：

  1. 必须同时保证表达式A和表达式B都符合左侧数据类型的要求；
